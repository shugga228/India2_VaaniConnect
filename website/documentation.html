Vaani Connect Project – Team Work and Goals Documentation

Fall 2025

This document summarizes the work completed by the team for Weeks 1–7, including planning, design, front-end development, documentation, coordination, and resource management.

Week 1 – [8/25/2025] – Team Work – Project Orientation

Team Work and Accomplishments:

  • Familiarized the team with Purdue EPICS structure, workflow, and expectations.
  • Reviewed past project examples, resources, and documentation requirements.
  • Discussed potential project ideas and identified areas where team members’ prior experience could be leveraged.
  • Began preliminary planning for project timeline and deliverables.

Goals: Understand EPICS workflows and project expectations. Align team members on semester timeline and responsibilities. Identify preliminary project ideas and potential contributions.

Week 2 – [9/1/2025] – Team Work – App Selection and Initial Planning

Team Work and Accomplishments:

  • Decided to focus on the Vaani Connect app, leveraging prior app development experience.
  • Created initial flowcharts and diagrams representing app logic, navigation, and core functionality.
  • Researched web development and UI standards to guide early design decisions.
  • Began drafting initial wireframes for the app interface to align with project goals.

Goals: Finalize project selection. Develop initial app logic diagrams and wireframes. Establish design standards based on web/UI best practices.

Week 3 – [9/8/2025] – Team Work – Code Review and Project Setup

Team Work and Accomplishments:

  • Reviewed existing Vaani Connect web app code via GitHub to understand structure and features.
  • Selected React Native for Android development based on team expertise and original documentation requirements.
  • Set up initial React Native project structure with verified directories and files.
  • Refined UI wireframes collaboratively to match app logic.

Goals: Set up native app project structure. Begin front-end development preparation. Finalize UI wireframes for team alignment.

Week 4 – [9/15/2025] – Team Work – UI Refinement, Gantt Chart, and Financial Draft

Team Work and Accomplishments:

  • Collaborated on refining the app’s UI and wireframes.
  • Created a Gantt chart detailing project phases, milestones, and deadlines, assigning roles to each team member.
  • Drafted a financial plan for resource allocation and project budgeting.
  • Continued referencing web development and UI standards to ensure design compliance.

Goals: Finalize UI design and wireframes. Complete Gantt chart with assigned responsibilities. Prepare financial draft for project resource management. Ensure all design elements follow recognized web/UI standards.

Week 5 – [9/22/2025] – Team Work – Front-End Development (Translation Input Boxes)

Team Work and Accomplishments:

  • Began front-end development with the translation input boxes.
  • Adjusted to React Native’s TypeScript updates to maintain compatibility and functionality.
  • Shared code snippets and layouts among team members for collaborative feedback.

Goals: Implement core front-end components. Document and share code with the team. Refine wireframes to reflect front-end implementation.

Week 6 – [9/29/2025] – Team Work – Front-End Development (Output Section and Supported Languages)

Team Work and Accomplishments:

  • Completed the front-end output section and added supported languages.
  • Implemented placeholder outputs pending backend integration.
  • Reviewed UI designs collectively to ensure alignment with wireframes.
  • Improved development efficiency due to TypeScript familiarity.

Goals: Complete front-end interface for translations. Prepare placeholders for backend integration. Maintain design consistency with team-created wireframes.

Week 7 – [10/6/2025] – Team Work – Design Document, App Updates, and Website Maintenance

Team Work and Accomplishments:

  • Updated the design document and presentation collaboratively to reflect progress and planned features.
  • Added basic speech-to-text functionality, planning integration of the official API for regional accent accuracy.
  • Updated team website with current project status, documentation, and resources.
  • Continued reviewing web/UI standards to guide upcoming development and design adjustments.

Goals: Refine design documents and presentation for reviews. Integrate speech-to-text API in future development. Maintain website as a project hub. Ensure front-end and UI designs follow web/UI standards.
